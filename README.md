# Hi there! im Ebtehal, and this is my portfolio website.

A clean, responsive personal portfolio built to showcase my software engineering projects, skills, and experiences.

## 🚀 Tech Stack

* **Frontend:** React & TypeScript
* **Contact Form:** Web3Forms (for serverless email delivery)
* **Styling:** Tailwind CSS

---

## 🛠️ Running the Code

### Setup Instructions
1. Run `npm i` to install the dependencies.
2. Run `npm run dev` to start the development server.
